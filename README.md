# Instrucciones para correr el proyecto

1. Tener node.js (npm install) y MySQL instalados
2. Ejecutar el archivo BD/feriaLibro.sql, BD/funciones.sql y BD/datos.sql en ese orden
3. Configurar tu conexion a la base de datos en db.js
4. Ejecutar el comando: npm init -y (desde la consola en el directorio del proyecto)
5. Ejecutar el comando: npm install cors express mysql2 body-parser (desde la consola en el directorio del proyecto)
6. Ejecutar el comando: npm start (desde la consola en el directorio del proyecto)
